# geodata-spain

Georeferenced data from the Autonomous Communities and Provinces of Spain. The original data comes from te [Download Center of the CNIG](https://eur02.safelinks.protection.outlook.com/?url=https%3A%2F%2Fcentrodedescargas.cnig.es%2FCentroDescargas%2FdetalleArchivo%3Fsec%3D9000029&data=05%7C02%7Cgabriel.lopez%40ineco.com%7Caa608c2d10824452bc9908de38f6141b%7C24a632fe975448c6b0a6be004693446d%7C0%7C0%7C639010826723519927%7CUnknown%7CTWFpbGZsb3d8eyJFbXB0eU1hcGkiOnRydWUsIlYiOiIwLjAuMDAwMCIsIlAiOiJXaW4zMiIsIkFOIjoiTWFpbCIsIldUIjoyfQ%3D%3D%7C0%7C%7C%7C&sdata=VBNNEjXRt6RrmcmF91EAHzylWbudzMSZsIexeODMyFk%3D&reserved=0).

The data stored in this repository includes also the INE Official Autonomous Community and Province Codes defined in the [INE Codes Webpage](https://ine.es/daco/daco42/codmun/cod_ccaa_provincia.htm) as extra properties, and the names of the Autonomous Communitites and Provinces have been standardised and replaced by those defined there.

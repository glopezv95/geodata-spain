# geodata-spain

Georeferenced data from the Autonomous Communities and Provinces of Spain. The original data comes from te [Download Center of the CNIG](https://centrodedescargas.cnig.es/CentroDescargas/detalleArchivo?sec=9000029).

The data stored in this repository includes also the INE Official Autonomous Community and Province Codes defined in the [INE Codes Webpage](https://ine.es/daco/daco42/codmun/cod_ccaa_provincia.htm) as extra properties, and the names of the Autonomous Communitites and Provinces have been standardised and replaced by those defined there.
